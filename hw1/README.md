# Web Programming HW#1

## Run my code (Basically same as lecture)

### Prepare .env file
1. Copy .env.exaple and save it as .env. 
2. paste your MONGO_URL in it, and save the PORT=8000.

### Run!!!
1. $ cd backend
2. $ yarn
3. $ yarn start
4. Open frontend/index.html in browser
5. Join it!
